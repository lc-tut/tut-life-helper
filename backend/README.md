# 3D Campus Map - Backend

このディレクトリは、フロントエンドにキャンパスの建物データを提供するFastAPIバックエンドです。

## 起動方法

Anaconda環境（`uni_map`）がアクティブな状態で、以下のコマンドを実行します。

```bash
uvicorn main:app --reload
# または
python main.py
```

サーバーは `http://localhost:8000` で起動します。
APIのドキュメント（Swagger UI）は `http://localhost:8000/docs` で確認できます。

## データの変更方法

建物の情報や配置を変更する場合は、同じディレクトリにある `data.json` を編集してください。
配列の要素として複数の建物を定義できます。

### JSONの記述ルール（スキーマ）

#### 1. 建物オブジェクト (Building)
キャンパス上の1つの建物を表します。
- `id` (文字列) **必須**: 建物のユニークな識別子。他の建物と被らないようにしてください。
- `name` (文字列) **必須**: 建物の表示名。
- `position` (オブジェクト) **必須**: 3D空間上の配置座標。中心が `{ "x": 0, "z": 0 }` となります。
  - `x`: 横方向の位置。
  - `z`: 奥行き方向の位置。（3D空間ではY軸が高さとなるため、平面上の配置はXとZで指定します）
- `floors` (配列) **必須**: その建物の各階層の情報のリスト。

#### 2. 階層オブジェクト (Floor)
建物の1つの階を表します。
- `level` (数値) **必須**: 階の順番（1, 2, 3...）。この数値に基づいて下から上に積み上げられます。
- `name` (文字列) **必須**: 階の表示名（例: "1F", "地下1階"）。
- `details` (文字列) **必須**: 建物をクリック時にUIに表示される詳細情報（テナントや教室名など）。
- `rooms` (配列) *任意*: その階にある間取り（部屋やトイレなど）の情報のリスト。

#### 3. 部屋・設備オブジェクト (Room / POI)
階層をクリックした際に表示される、内部の小さな3Dボックスを表します。
- `id` (文字列) **必須**: 部屋のユニークな識別子。
- `type` (文字列) *任意*: 部屋の種類（プログラムで判別用）。
- `label` (文字列) *任意*: 部屋の名前。
- `color` (文字列) **必須**: 3D空間に表示されるボックスの色。カラーコード（例: `"#1e90ff"`, `"#ff69b4"`）で指定します。
- `position` (オブジェクト) **必須**: **その階層の中心を `{ "x": 0, "z": 0 }` としたときの相対座標**。
  - ※現在の建物のデフォルトサイズは幅・奥行きともに `4`（半径2）のため、`x` と `z` を `-2.0` 〜 `2.0` の範囲に設定すると建物の箱の中に収まります。
- `size` (オブジェクト) **必須**: ボックスの幅(`x`)と奥行き(`z`)のサイズ。

### データ構造の記述例

```json
[
  {
    "id": "bldg1",
    "name": "1号館",
    "position": {"x": -10, "z": -10},
    "floors": [
      {
        "level": 1, 
        "name": "1F", 
        "details": "学生ホール、カフェテリア",
        "rooms": [
          {
            "id": "t_men_1",
            "type": "toilet_men",
            "label": "男子トイレ",
            "color": "#1e90ff",
            "position": {"x": 1.2, "z": -1.2},
            "size": {"x": 0.8, "z": 0.8}
          }
        ]
      },
      {"level": 2, "name": "2F", "details": "講義室"}
    ]
  }
]
```

フロントエンドはここで設定された `position` に基づいて建物を配置し、`floors` の数だけブロックを積み上げます。
